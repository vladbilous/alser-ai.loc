# Проєкт Alser AI

Цей проєкт є статичним веб-сайтом, який підтримує запуск через Docker для зручного локального розгортання.

## 🛠 Технології, що використовуються

Сайт побудований на базових веб-технологіях без використання важких фреймворків чи серверної мови (як PHP чи Node.js):
- **HTML5** — для створення структури сторінок (файли `index.html`, `catalog.html`, `product.html`).
- **CSS3** — для стилізації та адаптивного дизайну (папка `css/`).
- **JavaScript (Vanilla JS)** — для інтерактивності (папка `js/`). Включає `main.js` для базових скриптів сайту та `chatbot.js` для логіки роботи чат-бота.
- **Nginx** (через Docker) — легкий та швидкий веб-сервер, який використовується для віддачі цих статичних файлів у браузер.

## ⚙️ Як працює сайт

Сайт є повністю статичним фронтендом. Це означає, що коли користувач відкриває адресу сайту, веб-сервер просто передає готові `.html`, `.css` та `.js` файли до його браузера. Вся логіка (наприклад, відкриття меню, робота слайдерів чи чат-бота) виконується безпосередньо в браузері клієнта (на стороні клієнта), а не на сервері.

Історично проєкт працював через **Open Server Panel (OSPanel)** (на що вказує наявність папки `.osp`), але тепер його також можна легко розгорнути на будь-якій системі за допомогою Docker, що робить його незалежним від локальних налаштувань комп'ютера.

## 🚀 Як запустити сайт за допомогою Docker

Щоб сайт працював, на вашому комп'ютері має бути встановлена програма [Docker Desktop](https://www.docker.com/products/docker-desktop).

### Крок 1: Увімкніть Docker Desktop
Знайдіть програму **Docker Desktop** у меню «Пуск» Windows та запустіть її.
Зачекайте кілька секунд або хвилин, доки іконка Docker в системному треї (внизу праворуч біля годинника) не покаже статус "Engine running" (зелений індикатор).

### Крок 2: Запустіть проєкт
Відкрийте термінал (PowerShell, Command Prompt або термінал у вашому редакторі коду) у кореневій папці цього проєкту (де лежить файл `docker-compose.yml`) і введіть таку команду:

```bash
docker-compose up -d
```
*Що робить ця команда: вона бере легкий веб-сервер Nginx, підключає до нього всі файли вашого сайту і запускає це все у фоновому режимі (прапорець `-d`).*

### Крок 3: Відкрийте сайт
Після виконання команди відкрийте ваш браузер і перейдіть за адресою:
👉 **[http://localhost:8080](http://localhost:8080)**

### Корисні команди

- **Зупинити сайт:**
  ```bash
  docker-compose down
  ```
  *(Зупиняє і видаляє контейнер, але не видаляє ваші файли сайту).*

- **Переглянути логи веб-сервера:**
  ```bash
  docker-compose logs
  ```

- **Перевірити, чи працює зараз сайт:**
  ```bash
  docker ps
  ```
