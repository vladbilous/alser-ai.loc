# Alser AI Telegram Bot

AI-бот консультант для сайту alser.ua, який відповідає клієнтам 24/7.
Бот використовує Node.js, `node-telegram-bot-api` та OpenRouter API (модель `google/gemini-2.0-flash-exp`).

## Вимоги
- [Node.js](https://nodejs.org/) (рекомендовано версію 18 або вище)

## Встановлення залежностей

1. Перейдіть до папки з ботом у терміналі (якщо ви ще не в ній):
   ```bash
   cd telegram-bot
   ```
2. Встановіть необхідні залежності:
   ```bash
   npm install
   ```

## Налаштування оточення (.env)

1. Скопіюйте файл `.env.example` та перейменуйте його на `.env`. Або просто створіть новий файл `.env` та скопіюйте туди рядки з `.env.example`.
2. Відкрийте файл `.env` та вставте ваші ключі:
   ```env
   TELEGRAM_TOKEN=ваш_токен_телеграм_бота
   OPENROUTER_KEY=ваш_ключ_openrouter
   ```
   - **TELEGRAM_TOKEN** можна створити та отримати у [@BotFather](https://t.me/BotFather) в Telegram.
   - **OPENROUTER_KEY** можна згенерувати на сайті [OpenRouter](https://openrouter.ai/keys).

## Запуск бота

Щоб запустити бота, виконайте команду:
```bash
node index.js
```

Після успішного запуску ви побачите повідомлення в консолі:
`Бот успішно запущений і готовий до роботи...`

## Функціонал та структура
- **`index.js`** — головний файл проекту. Бот працює через polling (webhook не потрібен).
- Ігнорує пусті повідомлення та обробляє виключно текст (зображення чи стікери будуть проігноровані).
- У разі помилки API надсилає клієнту повідомлення: *"Сервіс тимчасово недоступний"*.
- Веде детальне логування вхідних повідомлень, помилок та відправку відповідей у консоль.
